\subsection{Julia как язык разработки решателя}
\label{julia_intro}

При планировании программного проекта имеется огромный выбор языков программирования. При выборе языка программирования нужно учитывать множество факторов, такие как производительность и безопасность приложения или количество строк кода.

Перед началом решения любой задачи разработки ПО следует этап подготовительной работы. Выбор языка является важнейшей частью этого этапа.

При выборе языка программирования для этого проекта следует учитывать следующие факторы.

\begin{itemize}
\item Целевая платформа.
\item Гибкость языка.
\item Время исполнения проекта.
\item Производительность.
\item Поддержка и сообщество.
\end{itemize}

Проанализируем по выбранным факторам новый быстро развивающийся язык для технических расчетов Julia.

Язык программирования Julia -- это высокоуровневый высокопроизводительный свободный язык программирования с динамической типизацией, созданный для математических вычислений\cite{Balbaert2016}. Эффективен также и для написания программ общего назначения. Синтаксис языка схож с синтаксисом других математических языков (например, MATLAB и Octave), однако имеет некоторые существенные отличия. Julia написана на Си, C++ и Scheme. В стандартный комплект входит JIT-компилятор на основе LLVM, благодаря чему приложения, полностью написанные на языке, практически не уступают в производительности приложениям, написанным на статически компилируемых языках вроде Си или C++. Большая часть стандартной библиотеки языка написана на нём же, и содержит лучшие открытые библиотеки C и Fortran для линейной алгебры, генерации случайных чисел, обработки сигналов и работы со строками. В распоряжении имеются умный компилятор, распределенное параллельное исполнение, численная точность и большая библиотека математических функций. Также язык имеет встроенную поддержку большого числа команд для распределенных вычислений. Сообщество разработчиков Julia поставляет большое количество внешних пакетов через встроенный пакетный менеджер. По аналогии с популярной утилитой Jupyter, существует многофункциональный браузерный графический интерфейс IJulia.

Ниже перечислены некоторые из встроенных возможностей Julia\footnote{Подробнее на http://julialang.org.}.

\begin{itemize}
\item Мультиметод: обеспечивает возможность определять поведение функции в зависимости от типа передаваемых аргументов.
\item Динамическая типизация.
\item Хорошая производительность, сравнимая со статичиски компилируемыми языками такими, как C.
\item Встроенная система управления пакетов.
\item Макросы и другие возможности метапрограммирования.
\item Архитектура, специально спроектированная для параллельных и распределенных вычислений.
\item Сопрограммы: легковесные "зеленые" потоки.
\item Возможность определять пользовательские типы, не уступающие в скорости и удобстве встроенным.
\item Автоматическое создание эффективного специализированного кода для различных типов аргументов.
\item Элегантные и расширяемые преобразования для числовых и других типов.
\item Поддержка Unicode, включая (но не ограничиваясь) UTF-8.
\item Лицензия MIT: бесплатность и открытость исходного кода.
\end{itemize}

Основанный на LLVM JIT-компилятор Julia и дизайн самого языка позволяют ему по производительности приблизиться и часто сравняться с производительностью языка C\cite{Kwon2016}.

Julia не заставляет пользователя следовать какому-либо определенному стилю параллелизма\cite{Tate2014}. Вместо этого, в распоряжении имеется множество ключевых строительных блоков для распределенных вычислений, которые позволяют использовать различные стили и добавлять новые.

JuliaBox позволяет запускать ноутбуки IJulia в контейнерах-песочницах Docker. Это открывает простор для полностью облачной обработки, включая управление данными, редактирование и обмен кодом, выполнение, отладки, анализа и визуализации\cite{Rohit2016}. Цель этого проста -- перестать беспокоиться об администрировании машин и управлении данными и сразу перейти к решению задачи.

Таким образом, видно, что Julia имеет хорошие показатели по всем рассматриваемым факторам. Такой выбор поможет создать компактное, простое в отладке, расширении, документировании и исправлении ошибок решение.
